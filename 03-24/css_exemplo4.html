Row

Um widget que exibe seus filhos em uma linha horizontal (ou disposição horizontal).

Para fazer com que um filho se expanda para preencher o espaço horizontal disponível, envolva-o em um widget Expandido.

O widget Row (linha) não possui scroll (e, em geral, é considerado um erro ter mais filhos em uma Linha do que o espaço disponível). Se você tiver uma linha de widgets e quiser que eles possam rolar caso não haja espaço suficiente, considere usar uma ListView.

Para uma variante vertical, consulte Column.

Se você tiver apenas um filho, considere usar Align ou Center para posicioná-lo.
