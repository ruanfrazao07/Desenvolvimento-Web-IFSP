2.2. Working with Integer values

int variable store a number without a decimal point. 

void main() {
          int minhaVariavel = 35;
        
          print(minhaVariavel);
        }

int is a subtype of num. 

void main() {
          // tipo rotulo = valor;
          int minhaVariavel = 35;
        
          print(minhaVariavel);
        }

int is non-nullable 

void main() {
          // tipo rotulo = null;
          int minhaVariavel;
        
          print(minhaVariavel); // Error: Non-nullable variable 'minhaVariavel' must be assigned before it can be used.
        }

2.3. Working with double values

double variable store a number including a decimal point. 

void main() {
          double precoMinimo = 2.99;
        
          print(precoMinimo);
        }

null 

void main() {
          // type label = null;
          double precoMinimo;
        
          print(precoMinimo); // Error
        }

2.5. Working with Strings

String variable store a sequence of characters. 

void main() {

          print(descricao);
        }

Strings use UTF-16 (support emojis)

2.6. Printing information to the console

Use a print statement to display information from an application. 

void main() {
          String nomeCliente = 'Fulano de Tal';
          int idadeCliente = 22;
        
          print('Nome: ');
          print(nomeCliente);
          print('Idade: ');
          print(idadeCliente);
        }

Use the $ character to reference a variable in a print statement.

$ replace label with variable value 

void main() {
          String nomeCliente = 'Fulano de Tal';
          int idadeCliente = 22;
        
          print('Nome: $nomeCliente');
          print('Idade: $idadeCliente');
        }
